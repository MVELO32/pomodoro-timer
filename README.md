# Pomodoro Timer

A modern Pomodoro timer built in Python using Tkinter and ttkbootstrap.  

## Features
- Dark theme UI
- Session number display
- Circular progress meter
- Sound alerts & desktop notifications
- Works as a standalone Linux app

## How to Run
1. Clone the repo:
\`\`\`bash
git clone git@github.com:MVELO32/pomodoro-timer.git
\`\`\`
2. Create a virtual environment and install dependencies:
\`\`\`bash
cd pomodoro-timer
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
\`\`\`
3. Run the app:
\`\`\`bash
python pomodoro.py
\`\`\`
